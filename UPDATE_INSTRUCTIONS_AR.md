# طريقة تحديث التطبيق على GitHub

1. افتح مستودع Aladdin في GitHub.
2. اضغط Add file ثم Upload files.
3. ارفع جميع ملفات النسخة الجديدة.
4. وافق على Replace للملفات الموجودة.
5. اضغط Commit changes.
6. انتظر دقيقة ثم افتح التطبيق وحدّث الصفحة.
7. على iPhone: أغلق التطبيق من مبدّل التطبيقات ثم افتحه من جديد.

ملاحظة: رسائل Telegram التلقائية لا يمكن أن تعمل من تطبيق محلي فقط عندما يكون التطبيق مغلقاً. تحتاج خدمة خلفية مجدولة أو خادمًا.
