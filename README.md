# Rental Manager PWA

تطبيق إدارة إيجارات يعمل على iPhone كتطبيق PWA.

## التشغيل
يجب رفع محتويات المجلد على استضافة HTTPS مجانية مثل GitHub Pages أو Cloudflare Pages.
بعد فتح الرابط في Safari على iPhone:
1. اضغط زر المشاركة.
2. اختر Add to Home Screen / إضافة إلى الشاشة الرئيسية.
3. افتح التطبيق من الأيقونة.

## التخزين
البيانات محفوظة محلياً في Safari على الجهاز.
استخدم زر النسخة الاحتياطية بشكل دوري واحفظ الملف في iCloud Drive.

- منع وجود أكثر من مستأجر نشط في الغرفة
- تسجيل الدفعات
- تنبيهات انتهاء العقود
- عربي/إنجليزي
- نسخ احتياطي واستعادة
